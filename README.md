## 🚀 Projetos em Destaque

### 🔹 ONG CAMM4 — Sistema em Produção

📌 Sistema completo de gestão para ONG **em uso real**, desenvolvido para substituir processos manuais e centralizar dados administrativos.

**Impacto real**
- Utilizado por **5 usuários ativos** (Diretor, Gestor e Voluntários)
- Digitalização de processos internos da ONG
- Centralização e rastreabilidade de dados

**Tecnologias**
- NestJS + Prisma + PostgreSQL (Neon)
- JWT + Refresh Token (com rotação)

**Funcionalidades**
- Autenticação segura e controle de acesso (RBAC)
- Sistema de auditoria com logs de ações
- Geração de relatórios em PDF
- Gestão completa de usuários, crianças, frequência e doações

**Arquitetura**
- Estrutura em camadas (Controller, Service, ORM)
- Separação de responsabilidades
- Sistema modular e escalável

🔗 Repositório:  
[ONGCAMM4](https://github.com/RickelmeDSC/ONGCAMM4)
