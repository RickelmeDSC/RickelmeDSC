## 🔐 Destaques Técnicos

- Autenticação com JWT + Refresh Token (rotação)
- Controle de acesso (RBAC)
- Auditoria de ações no sistema
- Rate limiting e proteção de sessão
- API estruturada e documentada (Swagger)

- Logging centralizado para auditoria
- API-first (backend desacoplado do frontend)
